Renesse Aan Zee widget plugin.

== Beschrijving ==

Renesse Aan Zee is een WordPress plugin die een widget biedt voor het weergeven van evenementen en activiteiten. Perfect voor lokale ondernemers en gemeenschapswebsites.

== Installatie ==

1. Upload de map `renesse-aan-zee` naar de `/wp-content/plugins/` directory.
2. Activeer de plugin via het 'Plugins' menu in WordPress.
3. Ga naar 'Instellingen' -> 'Renesse widget' om de widget positie aan te passen op je website.
4. Configureer de widgetinstellingen indien nodig.

== Veelgestelde Vragen ==

= Hoe voeg ik evenementen en activiteiten toe? =

De evenementen en activiteiten worden willekeurig ingeladen vanuit http://renesseaanzee.nl/

= Kan ik het uiterlijk van de widget aanpassen? =

Ja, je kunt de positie van de widget instellen.
